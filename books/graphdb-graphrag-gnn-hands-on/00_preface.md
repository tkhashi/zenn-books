## 本書について

本書は、**グラフDB・GraphRAG・GNN** という3つのグラフ関連技術をローカル環境で体験するハンズオンガイドです。

「グラフ」と聞いても、棒グラフや折れ線グラフのことではありません。ノードとエッジで関係性を表現する **グラフ構造** の技術群です。近年、知識グラフや RAG（Retrieval Augmented Generation）の文脈で急速に注目が高まっています。

3つの技術を **映画レコメンデーション** という共通テーマで繋ぎ、段階的に体験できるよう構成しました。

| 技術 | 本書での役割 |
|------|------------|
| **GraphDB（Neo4j）** | 映画・ユーザー・評価の関係をグラフとして保存・検索する |
| **GraphRAG** | 映画の概要テキストから知識グラフを構築し、自然言語で質問する |
| **GNN** | グラフ構造を学習して「あなたへのおすすめ映画」を予測する |

## 前提知識

- Python の基本的な読み書きができる
- コマンドライン操作（cd / ls 程度）に慣れている

機械学習・グラフ理論・データベースの事前知識は **不要** です。

## 動作環境

| 項目 | 要件 |
|------|------|
| PC | Mac（Apple Silicon: M1/M2/M3/M4） |
| メモリ | 16GB 推奨（8GB 可、ただし Docker メモリ設定要調整） |
| ストレージ空き | 5GB 以上 |
| Docker Desktop | Apple Silicon 版（インストール済みであること） |

## OpenAI API について

GraphRAG の章（Ch.06〜07）では OpenAI API を使用します。

- 使用モデル: `gpt-4o-mini`、`text-embedding-3-small`
- 推定コスト: **約 $0.025（約4円）** ※映画20件で構築した場合

API キーの取得方法は Ch.02 で説明します。
